## One-liner

**"Boil the Ocean, Together."**

## 3-tier 메시지

- **어제까지** — 인력·시간·기술력 비용이 너무 비싸 *"바다를 끓이지 마라(Don't boil the ocean)"* 가 정답이었다.
- **오늘** — AI 등장으로 그 셋이 0에 수렴. *한 사람이 나부터* 끓일 수 있는 조건이 처음 갖춰졌다.
- **내일** — 각자 끓이고, 시도와 결과를 나누면 *Together*. 함께 끓는다.

## 1장 인트로 (0:00–3:00, 3분)

- **핵심 메시지:** 플러터 서울 오거나이저 + 19년차 제너럴리스트(조선해양공학 중퇴 → 호기심 개발) + 카카오 출신 / 스타트업 공동창업자. 회사 이야기 X, **개인적 경험·생각**. 영어 One-liner를 *빈칸* → *정답*으로 두 단계 빌드.
- **대사 한 줄:** **"Boil the Ocean, ________"** → **"Boil the Ocean, Together."**

## 2장 Garry Tan 자기 모순 (3:00–5:30, 2.5분)

- **핵심 메시지:** 같은 사람·같은 매체, 16년 사이 정반대 메시지. Don't 풀이에 *자원 유한성*(인력·시간·돈) 한 줄을 깔아 5장 / 7장 회수 준비.
- **대사 한 줄:** "왜 같은 사람이 16년 만에 정반대를 말했을까요? — 그 답을 *함께* 찾아보겠습니다."

## 3장 AI의 등장 (5:30–10:30, 5분)

- **핵심 메시지:** 자기 모순의 *원인*을 시간순 4단계 사실로 못 박기. 결론 톤은 *"가치관이 바뀐 게 아닌, 세상·기술이 바뀌었다"*.
- **타임라인:** 2022.11 ChatGPT → 2023 GPT-4·Claude(+RAG/LangChain/VectorDB) → 2024 GPU 전쟁·Cursor(구독 모델·1.5x 생산성) → 2025 Claude Code·Agent 진화(Command→MCP→Skills→Sub-Agent)
- **대사 한 줄:** "이 4단계는 3년 반 동안의 이야기 — Garry Tan의 가치관이 바뀐 게 아닌, 세상이, 기술이 바뀌었다는 뜻입니다."

## 4장 7개 표준 개념 (10:30–16:30, 6분)

| 개념 | 1줄 정의 |
| --- | --- |
| Model | 사전 학습된 추론 시스템 — AI의 두뇌 |
| LLM | 자연어 입출력 모델 — 글로 답하는 모델 |
| Prompt | 모델에 1회 전달되는 입력 지시문 |
| Skill | 영구 저장된 절차 명세 — 반복 적용되는 매뉴얼 |
| Agent | 목표를 받아 자율 작동하는 상위 모드 |
| MCP | 외부 데이터·도구 연결의 표준 프로토콜 |
| Harness | Agent 실행 환경의 총체 — 권한·훅·설정의 합 |

- **3블록 (1+2+3분):** Model+LLM+Prompt / Skill+Agent / MCP+Harness
- **누적 학습 장치:** "SK하이닉스의 주가를 예측해서 알려줘" 시나리오 하나로 7개 개념을 블록 1→2→3에 걸쳐 쌓아올림. 블록 3에서 *Skill만으론 외부 데이터 접근 불가*를 드러내며 6-6 데모 **복선** 역할.
- **대사 한 줄:** "Harness는 7장에서 다시 회수합니다."

## 5장 Boil — 개인 문제 도입 (16:30–18:30, 2분)

- **핵심 메시지:** 자기 모순 답 = "이제 한 사람이 끓일 수 있다." *Together*는 7장에서 회수, 여기서는 **나부터** — 자기 자신의 가장 아픈 문제부터.
- **개인 사례:** 5년 한 종목 고집 → -2.5억 손절 → 5천만원 잔여 → "이걸 AI에게 시켜보면 어떨까?"
- **대사 한 줄:** "Boil the ocean이 옳아진 이유는 — 이제 한 사람이 *나부터* 끓일 수 있기 때문입니다."
